Remarks

The handler is invoked on the peer listener thread. If it throws, then the connection is closed but the listener remains active. It is passed a duplex stream, the local address and the remote address.

To stop listening, the cancel parameter must be supplied and then use the Cancel() method.

For socket based transports (tcp or upd), if the port is not defined or is zero an ephermal port is assigned.
