Finding a peer

If a peer's ID is known but not it's addresses, then Dht.FindPeerAsync can be used. It will ask other peers for the details of the unknown peer.

var peerId = new MultiHash("QmdpwjdB94eNm2Lcvp9JqoCxswo3AKQqjLuNZyLixmCxxx");
peer = await dht.FindPeerAsync(peerId);

Algorithm

A DHT distributed query is created that iteratively asks the closest peers to the unknown peer for its details.
